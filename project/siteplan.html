Dice Roller Siteplan

1. Site Name

The site name will be "D&D Dice Roller". It's very simple and to the point.
It's a dice roller to be used for table-top roleplaying games, but I'll call it "D&D Dice Roller" because D&D is the most common TTRPG

2. Site Purpose

As previously stated, it will be a dice-rolling application for TTRPGS.
